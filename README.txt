Данные файлы созданы в условии, если вы собираете полноценную систему защиты с помощью микропроцессорной техники, включающую:
Arduino Leonardo - 1 шт.
Arduino UNO — 3 шт.
Герконы — 2 шт.
Магниты — 2 шт.
RFID MFRC522 - 2 шт.
Модуль SD для Arduino - 1 шт.
Часы реального времени для Arduino - 1 шт.

Инструкция по сборке:
1. Установите на свой компьютер компилятор Arduino IDE. https://www.arduino.cc/en/software

2. Подключите к компьютеру Arduino Leonardo. Откройте файл leonardo.ino. 
   Скомпелируйте его с помощью установленной программы и загрузите в плату.
3. Подключите к компьютеру Arduino UNO. Откройте файл gercon1.ino. 
   Скомпелируйте его с помощью установленной программы и загрузите в плату.
4. Подключите к компьютеру вторую Arduino UNO. Откройте файл gercon1.ino. 
   Скомпелируйте его с помощью установленной программы и загрузите в плату.
5. Подключите к компьютеру третью Arduino UNO. Откройте файл chernSmor.ino. 
   Скомпелируйте его с помощью установленной программы и загрузите в плату.

6. На первой плате Arduino UNO свяжите порт 2 с портом 2 на Arduino Leonardo, замыкая на GND. 
   На второй плате Arduino UNO свяжите порт 2 с портом 3 на Arduino Leonardo, замыкая на GND. 
   Установите геркон на порт 3 и А0 на первой плате, замыкая на GND.
   Установите геркон на порт 3 и А0 на второй плате, замыкая на GND.
7. К третьей плате Arduino UNO подключите Модуль SD для Arduino, RFID MFRC522, Часы реального времени для Arduino согласно портам и рекомендациям, указанным в коде программы.
   Вставьте карту памяти, где будет находится база данных всех пользователей.
   Подключите плату Arduino UNO через порт 6 к порту 4 на Arduino Leonardo, замыкая на GND.

Система собрана!

Администратор

Приложите к считывателю "RFID MFRC522" ключ. Это будет картой администратора. 
Администратор может добавить нового пользователя, приложив свою карту, а затем карту нового пользователя.
Администратор может удалить пользователя, приложив свою карту дважды, а затем пользователя.
Вы можете расширить количество администраторов, изменив значение 1 на другое в коде программы chernSmor.ino во вкладке Administrator в первой строке.
  НАПРИМЕР: Изменив значение с 1 на 2 после компиляции и загрузки на плату первые 2 карты, поднесенные к считывателю "RFID MFRC522", будут зафиксированы как администраторские.

Герконы

Вы можете расширить количество герконов, изменив значение 1 на другое в коде программы gercon1.ino в первой строке.
Установите геркон на порт 3 и А0 на новой плате Arduino UNO, замыкая на GND.

База данных

На карте памяти будут записаны все пользователи. При желании можно подписать имя каждого пользователя на карте памяти рядом с ID карты.
В дальнейшем их имена будут отображаться на карте памяти.
В базе данных хранится время входа и выхода каждого пользователя, а также время записи и удаления пользователей администратором.
